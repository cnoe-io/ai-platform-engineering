What Should "Manager" Mean?

OpenFGA does not have a single global Manager role. We need to choose how the simulator constructs the simulated subject.

A

Role Profile Template

"Manager" means a canned permission profile, useful for design demos and policy testing before assigning a real user.

B

Concrete User or Team

"Manager" means select an actual user/team and evaluate their real OpenFGA relationships. Best for support and audits.

C

Both Modes

Start with real user/team simulation, then add role templates for planned policy changes and demos.
