# Changes & Features

Documentation of significant changes, features, and architectural decisions for the AI Platform Engineering project. Each doc is dated and listed in the **sidebar** (left).

## Status

- **In-use** — Implemented and actively used
- **Proposed** — Planned or partially implemented
- **Abandoned** — Deprecated or superseded

## Adding a change doc

1. Create a file: `YYYY-MM-DD-short-descriptive-name.md`
2. Start with a level-one heading: `# Title of the change`
3. Optionally add **Status** and **Category** at the top of the body.

New files appear automatically in the sidebar with the date in the title.
