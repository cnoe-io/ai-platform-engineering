> Released: 2026-06-30
> Chart: `oci://ghcr.io/cnoe-io/charts/ai-platform-engineering:0.5.32`
> Previous release: 0.5.31

## Highlights

0.5.32 is a targeted patch with two fixes: the AgentGateway config-bridge now correctly derives the ext_authz host from the Helm release name instead of a hardcoded value, and a broken cross-link in the architecture docs is repaired.

<!-- truncate -->

## Bug Fixes

- **AgentGateway ext_authz host is now release-aware**: The config-bridge was using a hardcoded hostname for the ext_authz endpoint, which broke deployments where the Helm release name differed from the default. The host is now derived from the release name at runtime. ([#2090](https://github.com/cnoe-io/ai-platform-engineering/pull/2090))
- **Broken cross-link in architecture/gateway.md**: A relative link in the gateway architecture doc resolved to a non-existent path and caused a build warning. The link is corrected. ([#2075](https://github.com/cnoe-io/ai-platform-engineering/pull/2075))

## Breaking Changes

No breaking changes. Drop-in upgrade from 0.5.31.
