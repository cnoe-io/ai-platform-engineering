## [0.0.2] — 2026-03-16

### Changed

- Consolidated all specs and ADRs into `docs/docs/specs/<###-feature-name>/` for automatic Docusaurus publishing

- `.specify/specs` is now a symlink to `docs/docs/specs/`

- Updated CONSTITUTION, ARCHITECTURE, SPECS, README to reference `docs/docs/specs/` as canonical location
