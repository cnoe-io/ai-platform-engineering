Cloud Control Framework
¶
Cloud Control Frameworks (CCF)
are a set of Cisco Security controls that align with industry security requirements like SOC2, ISO, etc.
Here is the list of CCFs categorized and prioritized based on who, i.e. Platform, Product, Both or 3
rd
Party, owns the responsibility of meeting the security control.
