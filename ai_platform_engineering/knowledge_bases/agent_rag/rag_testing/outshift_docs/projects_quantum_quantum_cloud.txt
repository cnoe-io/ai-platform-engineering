Set up kubeconfig
¶
These instructions are specifically for MacOS and the EKS cluster eks-quantum-1.
1. Required information:
   1. Cluster name: eks-quantum-1
   2. AWS Region: us-west-2
2. Log in to the AWS CLI with the quantum role for 380642323071 (eticloud scratch)
3. Run:
`aws eks update-kubeconfig --name eks-quantum-1 --alias eks-quantum-1 --region us-west-2`
Use kubectl
¶
To use kubectl, you must first be logged in to the the AWS account via the command-line. Once you have logged in run:
kubectl config use-context eks-quantum-1
