SCF-IAM-02.02
monitoring
Restrict access to monitoring system to Cisco internal network
¶
Target Date: FY??Q?
¶
Risk(s) Addressed
¶
Production monitoring system is publicly accessible from the Internet without Cisco VPN connection and may be vulnerable to man-in-the-middle attacks or zero-day vulnerabilities in the third-party authentication system
Description
¶
Make monitoring system (MCOM) only accessible from Cisco internal network.
Definition of Done
¶
Monitoring system (MCOM) used by ET&I SRE team only accessible from Cisco internal network.
Relevant Controls
¶
IAM-02.02
Objective Status: Not Started
