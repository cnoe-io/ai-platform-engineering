Log in to correct AWS account and IAM role using
SAML
Go to the
EC2 web console
Please be aware that EC2 resources, amongst others, are available in regions. If you do not see your resource immediately, double-check that you are in the correct region in the upper-right of the page.
Click on
Instances
.
Click the checkbox next to the instance you want to start or stop.
Click the
Instance State
dropdown in the upper right.
Select
Start instance
or
Stop instance
as required.
