ET&I URL Naming Conventions
¶
URL Schema:
¶
<1. App name>.<2. Cisco Internal/CIDR restricted (Optional)>.<3. Cluster Color>.<4. Deployment Stage (Optional)>.<5. Domain name>
URL Schema Explanation
¶
AppName: Name of the application
Example:
maestro, fednview, dbp
Cisco Internal/CIDR Restricted (Optional)
Example:
int
Cluster Color
Example:
blue/green
Deployment Stage (Optional)
Example:
dev, stage, prod
Domain Name
Example:
eticloud.io
Examples
¶

https://maestro-api.int.blue.dev.eticloud.io
