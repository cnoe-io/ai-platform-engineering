[Optional] Add CODEOWNERS
¶
Clone the target repo
git clone https://wwwin-github.cisco.com/eti/sre-pipeline-library
Add CODEOWNERS
cd <target-repo>
mkdir -p .github
echo "# Following users are automatically added to PRs
*    @sraradhy @dneduva @petersz ahismail @imangass @subbaksh @haskalpa" > .github/CODEOWNERS
Commit and Merge to the default code
Add branch protection
¶
Navigate to repo settings
Select
Branches
tab
Select protected branch settings
Save settings
