Introduction
¶
Source Code Management (SCM) is a generic term used for source code version control.
[1]
[2]
Git
/
GitHub
is the preferred SCM for most developers in ET&I (and Cisco at large).
GitHub Enterprise
¶
At present, ET&I projects are hosted on either of two GitHub Enterprise (GHE) platforms:
Our internal/on-prem GitHub Enterprise Server (GHES) instance is available at
wwwin-github.cisco.com
, or
GitHub Enterprise Cloud (GHEC) instance available at
github.com
Within each instance, many GitHub "organizations" exist to logically separate various use cases. See the table below for more information.
