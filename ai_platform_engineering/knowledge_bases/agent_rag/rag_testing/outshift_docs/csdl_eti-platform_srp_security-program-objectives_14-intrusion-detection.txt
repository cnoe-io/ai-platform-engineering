IVS-01.01
Assessment and documentation of security tools and mechanisms used by ET&I Platform
¶
Target Date: FY??Q?
¶
Risk(s) Addressed
¶
Possible gaps not covered by Intrusion Detection Systems may inhibit the detection and effective reaction to malicious activity
Description
¶
Assess and document the different intrusion, malicious activity, and malware detection and protection systems on the different levels (cloud provider, network, host, container, etc) of the ET&I Platforms infrastructure and identify the possible gaps.
Definition of Done
¶
The ET&I SRE team has a comprehensive documentation and good understanding about the security tools and mechanisms that are provided by other Cisco teams. The document includes that gaps (if any) that should be covered.
