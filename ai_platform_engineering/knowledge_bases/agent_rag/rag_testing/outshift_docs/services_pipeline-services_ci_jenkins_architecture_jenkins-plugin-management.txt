Jenkins Plugin Management
¶
Jenkins Plugins extend the funtionality of Jenkins by allowing users to automate pipelines and workflows with various new capabilities
Plugins are also a source of security vulnerabilities if not patched appropriately
Installing/Updating Plugins
¶
Jenkins plugins are installed at instance level, hence SRE team is the gate keeper for installing plugins on
ETI Jenkins
instance
Cisco EngIT CI team installs/updates a default set of plugins
Installing/Updating plugins, in some cases, requires restarting Jenkins instance.
Note:
Plugin installation downtime needs to be communicated on
ETI Platform Status
space
