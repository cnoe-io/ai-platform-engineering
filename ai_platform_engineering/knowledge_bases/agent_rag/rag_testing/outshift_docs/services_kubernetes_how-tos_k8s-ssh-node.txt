K8s Node SSh
¶
Note that using this mechansim will create security alerts in Falco, the STO-mandated security scanning tool.
Please consider using our standard bastion hosts instead.  If you do use this mechansim, it is your responsibility to look for the resulting alert (sent via email) and resolve the alert with a comment indicating what what being done.
