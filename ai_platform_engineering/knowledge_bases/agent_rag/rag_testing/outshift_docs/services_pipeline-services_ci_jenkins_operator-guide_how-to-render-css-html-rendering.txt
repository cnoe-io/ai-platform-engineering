Solution Summary
¶
Update the Jenkins Content Security to allow Jenkins to render HTML/CSS
Navigate to
Manage Jenkins
->
Manage Nodes and Clouds
->
Master
->
Script Console
Execute following command
System.setProperty("hudson.model.DirectoryBrowserSupport.CSP","")
Verify the URLs are rendered correctly
