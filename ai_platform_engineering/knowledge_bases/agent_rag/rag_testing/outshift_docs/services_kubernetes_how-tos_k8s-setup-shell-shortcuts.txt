Pre-quisites
¶
Setup aws console with
duo-sso
Setup gcloud by installing the
following
Make sure you have the latest
gke auth plugin
by running:
gke-gcloud-auth-plugin --version
You may install it by running (outside of yout Google Cloud CLI install root folder):
gcloud components install gke-gcloud-auth-plugin

¶
Install k8s-shell-helper
Usage
¶
Login to P3 cluster
export VAULT_TOKEN=<foo>
p3 eks-gitops-1

defaults: region(us-east-2)

eks eks-prod-2 eticloud us-west-2

defaults: project(eticloud) zone(us-east1-a)
gke test-cluster
