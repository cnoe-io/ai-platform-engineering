# Azure DevOps MCP Server

MCP (Model Context Protocol) server for Azure DevOps.

## Features

- Manage projects
- Manage pipelines
- Manage repositories
- Manage work items
- Manage builds
- Manage pull requests

## Usage

```bash
mcp-azuredevops
```

## Environment Variables

- `AZDO_ORG_URL`: Azure DevOps organization URL
- `AZDO_PAT`: Azure DevOps Personal Access Token
