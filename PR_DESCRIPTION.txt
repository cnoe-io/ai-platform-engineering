# Description

This PR adds 3 new builtin tools to Dynamic Agents and restructures the API for better organization.

## New Builtin Tools

- **`current_datetime`** - Returns the current date/time in a specified timezone (default: UTC)
- **`user_info`** - Returns the authenticated user's email, name, and groups
- **`sleep`** - Pauses execution for a specified duration (configurable max, default 300s)

All new tools are enabled by default. The existing `fetch_url` tool remains disabled by default.

## API Restructuring

- **New endpoint:** `GET /api/v1/builtin-tools` - Dynamic discovery of available builtin tools with metadata
- **Moved endpoint:** `/api/v1/agents/models` → `/api/v1/llm-models` - Better REST semantics

## Additional Improvements

- **Userinfo caching:** 10-minute cache for OIDC userinfo to reduce auth server load
- **OIDC name parsing:** Support for non-standard `fullname`, `firstname`, `lastname` fields
- **Config seeding:** Stale config-driven entities are now cleaned up on seed
- **Default tools fix:** Default-enabled tools are now properly persisted to MongoDB
- **Conversation ID display:** Added to UI's Agent tab for debugging
- **Logging consistency:** Renamed "session" to "conversation" in logs (`conv=` prefix)
- **Documentation:** Added AgentRuntime deep-dive section to ARCHITECTURE.md
